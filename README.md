# Lui

A minimalistic Love2D GUI library, written entirely in Love2D.

## Features

- [x] Core
    - [x] Pane
    - [x] Window Stage
    - [x] Input
- [ ] Layouts
    - [x] Grid
    - [x] Container
    - [x] StackContainer
    - [ ] HStackContainer
    - [ ] VStackContainer
    - [ ] ScrollContainer
- [ ] Widgets
    - [x] ColorRect
    - [x] Window
    - [x] Label
    - [ ] Button
    - [ ] Canvas
