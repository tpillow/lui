# Lui

A minimalistic Love2D GUI library, written entirely in Love2D.

## Features

- [x] Core
    - [x] Pane
    - [x] Window Stage
    - [x] Input
- [p] Layouts
    - [x] Grid
    - [x] Container
    - [x] StackContainer
    - [ ] HStackContainer
    - [ ] VStackContainer
    - [ ] ScrollContainer
- [p] Widgets
    - [x] ColorRect
    - [x] Window
    - [p] Label
    - [ ] Button
    - [ ] Canvas
