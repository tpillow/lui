# Lui

A minimalistic Love2D GUI library, written entirely in Love2D.

## Features

- [ ] Core
    - [x] Pane
    - [x] Window Stage
    - [x] Input
    - [ ] Global Style System
    - [ ] JSON UI Save/Load
- [ ] Layouts
    - [x] Grid
    - [x] Container
    - [x] StackContainer
    - [ ] HStackContainer
    - [ ] VStackContainer
    - [ ] ScrollContainer
- [ ] Widgets
    - [x] ColorRect
    - [x] Window
    - [x] Label
    - [x] Canvas
    - [ ] ComboBox
    - [ ] Menubar
    - [ ] Button
